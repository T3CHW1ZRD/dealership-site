# Little Guys Motors

A simple, mobile-first website for **Little Guys Motors**, a local used-car dealership.

Static single-page site (`index.html`) — no build step, no dependencies.

## Sections
- Black GMC hero
- Cars for sale (with full-detail view + "Request this car")
- Monthly payment **estimator** (estimate only — no financing offered)
- Services: oil, brakes/tires/repairs, detailing, and at-home pickup & return
- Sell your car (schedule a meeting)
- Contact

## Notes
- Address, phone, dealer license #, and email are **placeholders** — replace before launch.
- Vehicle photos are CC-licensed demo images (Wikimedia Commons); swap in real lot photos.
- Contact/sell forms are front-end demos — wire to email/Formspree or a backend to receive submissions.
